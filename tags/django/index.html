Django 本文主要是记录一些Django的初始概念。适用于第一次接触Django或者好久没碰Django后重新捡起来的复习。 新建Django Project django-admin startproject project_name 新建Django App python manage.py startapp app_name Django Project vs Django App 初学Django这两个概念我有点懵，Gin或者Flask是没有类似Django里类似的划分的。我目前对Django Project vs Django App的理解是下面这样的。 首先是Django官网的解释： An app is a web application that does something – e.g., a blog system, a database of public records or a small poll app. A project is a collection of configuration and apps for a particular website. A project can contain multiple apps. An app can be in multiple projects.
